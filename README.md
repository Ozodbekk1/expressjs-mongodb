API Endpoints
| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | /api/v1/create/user | Create a new user |
| GET | /api/v1/getallinfo | Get all user information |
| GET | /api/v1/getByID/:userID | Get a specific user by ID |
| GET | /api/v1/get/info | Get API status information |
| PUT | /api/v1/update/user/:_id | Update a user by ID |
| DELETE | /api/v1/delete/user/:_id | Delete a user by ID |
| POST | /api/v1/create/post | Create a new post |
| PUT | /api/v1/update/:_id | Update a post by ID |
| DELETE | /api/v1/delete/:_id | Delete a post by ID |
